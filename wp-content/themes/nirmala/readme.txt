== Description ==

Nirmala is the Indonesian word for Flawless.

Nirmala is a free WordPress theme built on top of the highly reputable Bootstrap 4 front-end framework.

Therefore, Nirmala theme is a complete solution for whatever contents you have.

It is as solid as the Bootstrap itself, optimized for readability, highly mobile friendly and with neat and balanced layout.

It is minimal, lightweight, yet multipurpose.


== Frequently Asked Questions ==

= How to set-up or modify content appearance? =

Nirmala theme is built on top of Bootstrap 4. Please refer to Bootstrap documentation to set up or modify your content.

https://getbootstrap.com/docs/

= How to change font? =

1. Go to Appearance > Customize

2. Add your own CSS style, in this case, font-family for body

Example:

body {
 font-family: Roboto, Arial, sans-serif;
}
